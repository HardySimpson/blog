这是一个使用 Jekyll 在 GitHub Pages 上运行的博客示例。

## 开始使用

1. 修改 [_config.yml](/_config.yml) 文件来配置你的网站
2. 创建新的文章放在 `_posts` 目录下
3. 自定义布局和样式

## Jekyll 简介

Jekyll 是一个静态网站生成器，特别适合用于 GitHub Pages。
