欢迎来到我的 GitHub Pages 博客

你现在看到的是一个完全静态的示例项目，不依赖任何后端或构建工具，直接由 GitHub Pages 托管。

你可以直接修改这个页面的 HTML 和 CSS，把它变成你自己的博客或个人主页。

查看 GitHub Pages 文档

如何开始写内容？

  • 编辑本页：修改 index.html，替换标题和正文。
  • 添加新页面：复制一份 about.html，修改为你想要的内容，然后在导航里加上链接。
  • 调整样式：在 assets/style.css 中修改颜色、字体和布局。

部署说明（简要）

  1. 把本仓库推送到 GitHub（例如仓库名为 blog）。
  2. 在 GitHub 仓库页面打开 Settings → Pages。
  3. 在 Source 里选择 Deploy from a branch，分支选 main，目录选 / (root)。
  4. 保存后等几分钟，GitHub 会给出你的 Pages 访问链接。
